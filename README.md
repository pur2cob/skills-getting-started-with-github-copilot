# skills-getting-started-with-github-copilot
Exercise: Get started using GitHub Copilot

## Greeting Program

This repository contains a simple JavaScript program that demonstrates basic functionality for learning GitHub Copilot. The program responds to various greetings and showcases how AI-assisted coding can help you write better code faster.

### Features

- Responds to different types of greetings ("hi", "hello", "hey", etc.)
- Case-insensitive input handling
- Random greeting generator
- Comprehensive test suite
- Easy to run and modify

### Usage

#### Running the Program

```bash
# Run the main greeting program
node greeting.js

# Or use npm script
npm start
```

#### Running Tests

```bash
# Run the test suite
npm test
```

### Files

- `greeting.js` - Main program with greeting functionality
- `test.js` - Test suite to verify program functionality
- `package.json` - Node.js project configuration

### Learning GitHub Copilot

This program is designed to help you practice using GitHub Copilot by:
1. Writing functions with clear, descriptive names
2. Adding helpful comments that guide Copilot suggestions
3. Creating test cases to verify functionality
4. Using common programming patterns that Copilot recognizes

Try modifying the code and see how GitHub Copilot suggests improvements and additions!
